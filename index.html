20/5/20

I've finished my Snake game! The code for a Snake game is actually more complicated then you might think. The complex bit is that you have to make an array that tracks and continously updates Snake's current position and it's previous position, and then draws the snake based on this. Every time the snake eats something, the array gets bigger and draws a bigger snake. You can adjust array size to change the size of Snake. If you make the array size begin as 1 space, at the start you get a 1 square sized snake. If it's 10 spaces, the snakes is 10 squares.
Anyway, to make my multicolor Snake I had to change the code. In my version, instead of the whole background being redrawn every frame, it only redraws the background at the last position in the snake array.
